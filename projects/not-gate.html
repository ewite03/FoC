Project 1: NOT Gate

The NOT gate is a basic logic gate that inverts the input. If the input is ON (1), the output is OFF (0), and vice versa.

Input

Output

1 ——— NOT ——— 1

NOT Gate Output:

1

How it works:

  • Input ON (1) → Output OFF (0)
  • Input OFF (0) → Output ON (1)

Click the ON or OFF buttons to see how the NOT gate inverts the input!
