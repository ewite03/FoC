Project 5: A Simple Machine Language

Learn about machine language instructions! Click on an instruction code to see what it does.

Op-code: 1

Operand: RXY

LOAD reg. R from cell XY.

This instruction loads a value from memory cell XY into register R. For example, if you use 156C, it loads from cell 6C into register 1.

All Machine Language Instructions

Op-code Operand Description
1 RXY LOAD reg. R from cell XY.
2 RXY LOAD reg. R with XY.
3 RXY STORE reg. R at XY.
4 0RS MOVE R to S.
5 RST ADD S and T into R. (2's comp.)
6 RST ADD S and T into R. (floating pt.)
7 RST OR S and T into R.
8 RST AND S and T into R.
9 RST XOR S and T into R.
A R0X ROTATE reg. R X times.
B RXY JUMP to XY if R = reg. 0.
